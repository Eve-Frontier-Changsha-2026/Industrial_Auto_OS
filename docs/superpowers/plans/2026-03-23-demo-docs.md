7. **Quick Start** — prerequisites + commands:

Prerequisites:
- Sui CLI >= 1.x
- Node.js >= 18
- Testnet SUI (faucet: `sui client faucet`)

Build contracts:
```bash
cd packages/industrial_core && sui move build
cd ../work_order && sui move build
cd ../marketplace && sui move build
```

Run tests:
```bash
cd packages/industrial_core && sui move test
cd ../work_order && sui move test
cd ../marketplace && sui move test
```

Start watcher:
```bash
cd watcher && npm install && cp config.example.yaml config.yaml
# Edit config.yaml with your package IDs and keypair
npm run build && npm start
```

Start frontend:
```bash
cd frontend && npm install
# Edit .env with your package IDs
npm run dev
```
